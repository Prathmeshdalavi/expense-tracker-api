# Expense Tracker REST API

##  Project Overview
Expense Tracker is a Spring Boot REST API application designed to manage daily expenses and income records. It allows users to perform CRUD operations on expenses and categories, helping track financial activities efficiently.

---

##   Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST APIs
- Postman (for API testing)

---

##   Architecture
The application follows layered architecture:

Controller → Service → Repository

- Controller Layer: Handles HTTP requests
- Service Layer: Contains business logic
- Repository Layer: Manages database operations

---

##   Features
- Add new expense
- Update expense
- Delete expense
- View all expenses
- Category management
- Monthly expense summary

---

##   Database
- MySQL database
- Entity relationships using JPA & Hibernate
- Proper table mapping and validations

---

##  API Testing
All APIs were tested using Postman with structured JSON responses.
